# 타입스크립트 설치하기

타입스크립트를 사용하고 싶어도 설치하지 않으면 말짱 도루묵이죠! 이번에는 여러분들의 프로젝트에서 타입스크립트를 설치하는 방법에 대해 알려드리도록 하겠습니다.

## 시작하기 전에

시작하기 전에, 타입스크립트를 설치하기 위해선 컴퓨터에서 자바스크립트를 실행 할 수 있게 해주는 NodeJS와 패키지 설치자인 npm(혹은 pnpm이나 yarn같은 것)이 필요합니다. 

## package.json이 프로젝트에 없는 경우?

다음과 같은 명령어를 프로젝트 폴더 내에서 실행합니다.
- npm의 경우: `npm init`
- pnpm의 경우: `pnpm init`
- yarn의 경우: `yarn init`

## npm으로 타입스크립트 설치

1. 터미널에 `npm install typescript`를 입력해 타입스크립트를 자바스크립트로 변환하는 컴파일러를 설치합니다.
2. 터미널에  `npx tsc --init`을 입력해 타입스크립트를 사용할 때 필요한 설정 파일인 tsconfig.json을 생성합니다.
3. (선택) 터미널에 `npm install ts-node`을 입력해 명령창에서 직접 타입스크립트 코드를 실행할 수 있게 해주는 도구를 설치합니다.

## pnpm으로 타입스크립트 설치

1. 터미널에 `pnpm add typescript`를 입력해 타입스크립트를 자바스크립트로 변환하는 컴파일러를 설치합니다.
2. 터미널에 `pnpm tsc --init`을 입력해 타입스크립트를 사용할 때 필요한 설정 파일인 tsconfig.json을 생성합니다.
3. (선택) 터미널에 `pnpm add ts-node`을 명령창에서 직접 타입스크립트 코드를 실행할 수 있게 해주는 도구를 설치합니다.

## yarn으로 타입스크립트 설치

1. 터미널에 `yarn add typescript`를 입력해 타입스크립트를 자바스크립트로 변환하는 컴파일러를 설치합니다.
2. 터미널에 `yarn tsc --init`을 입력해 타입스크립트를 사용할 때 필요한 설정 파일인 tsconfig.json을 생성합니다.
3. (선택) 터미널에 `yarn add ts-node`을 입력해 명령창에서 직접 타입스크립트 코드를 실행할 수 있게 해주는 도구를 설치합니다.

## +) npm으로 타입스크립트 전역 설치하기

타입스크립트를 전역으로 설치하게 된다면 모든 프로젝트에서 타입스크립트를 바로 사용 할 수 있으며, 무언가 아이디어를 떠올렸을때 그것을 바로 테스트해 볼수 있다는 장점이 있습니다. 설치하는 방법은 간단합니다. [npm으로 타입스크립트 설치](#npm으로-타입스크립트-설치)의 `npm install typescript`대신 `npm install -g typescript` 를 입력하면 됩니다.

## 글을 마치며

이렇게 타입스크립트를 설치하고 tsconfig.json까지 생성하는 방법을 알았습니다. 다음으론 어떻게 타입스크립트를 자바스크립트로 변환 할 수 있는지 알아보겠습니다.
