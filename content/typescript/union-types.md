# 유니온 타입이란?

이런 상황이 있지 않을까요? 아이디를 받아야 하는데, 여러 타입 중 어떤 타입으로 들어올지 모르는 상황 말입니다. 이럴때에는 어떻게 해결할 수 있을까요? 이렇게 해볼까요?

```typescript
function setId(id: any): void {
	id_array.push(id);
}
```

`any`를 사용하면 어떤 타입이든 다 받을 수 있겠죠. 하지만 만약 아이디 숫자를 변경하려면 문제가 생깁니다. 어떤 타입이 들어갈지 모르기 때문이죠. 만약 여기서 어떤 타입들이 들어오는지 알 수 있다면, 유니온 타입이 아주 좋은 솔루션일 것입니다.

## 유니온 타입

유니온 타입은 서로 다른 두 개의 타입을 이용해서 "이 타입 아니면 저 타입 들어간다" 라고 표시하는 타입입니다. 마치 자바스크립트의 `||` 나 파이썬의 `or`과 같은 느낌이죠. 유니온 타입은 이렇게 쓸 수 있습니다.

```typescript
매개변수: 타입1 | 타입2
```

또 두 개의 타입 말고도 여러 타입을 쓸 수도 있습니다.

```typescript
매개변수: 타입1 | 타입2 | 타입3 | 타입4
```

그럼 아까 전에 만들었던 함수를 유니온 타입을 통해 정의해볼까요? (참고로 이 함수에는 `string`과 `number`가 들어간다고 가정합니다.)

```typescript
function setId(id: string | number): void {
	id_array.push(id);
}
```

이렇게 하면 `any`를 사용하는 것 보다 훨씬 좋을 것입니다!

:::tip
유니온 타입을 사용하면 두 타입중 하나의 타입이 들어갑니다. 즉, 어떤 타입이냐에 따라 그것을 처리할 수 있어야 합니다. 그러니 if문과 typeof를 이용해 조건문을 짜는 것이 좋습니다.
:::
