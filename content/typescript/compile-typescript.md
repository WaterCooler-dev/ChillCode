# 타입스크립트를 자바스크립트로 변환하기

전 단계에서 설치한 `typescript`패키지, 기억하시나요? 이 패키지는 사실 타입스크립트 파일을 자바스크립트로 변환해주는 도구입니다. 웹브라우저에서는 타입스크립트를 실행하지 못하거든요. 그럼 어떻게 하면 이 도구로 타입스크립트를 변환 할 수 있을까요?

## 시작하기 전에

앞의 [2. 타입스크립트 설치하기](/typescript/install-typescript) 단계를 완료하셔야 타입스크립트를 컴파일 하실 수 있습니다. 아니면 시작하지도 못합니다!

## 컴파일러로 타입스크립트 변환하기

### 원하는 파일만 변환하기

터미널에

- 그냥 설치 한 경우: 
	- npm: `npx tsc <파일 이름>.ts`
	- pnpm: `pnpm tsc <파일 이름>.ts`
	- yarn: `yarn tsc <파일 이름>.ts`
- 글로벌로 설치한 경우: `tsc <파일 이름>.ts`

를 실행합니다.

(진짜 몰라서 말씀드리는거고, 이것 때문에 기분 나쁘셨다면 죄송합니다만 `<파일 이름>`이라고 적힌 부분은 타입스크립트 파일의 이름을 적으셔야 합니다! 아니면 전혀 실행되지 않습니다.)

이렇게 했을때 아무 오류가 없었다면, 축하드립니다! 성공적으로 타입스크립트 파일을 자바스크립트로 변환하셨습니다!

### 모든 파일 변환하기

터미널에

- 그냥 설치 한 경우: 
	- npm: `npx tsc`
	- pnpm: `pnpm tsc`
	- yarn: `yarn tsc`
- 글로벌로 설치한 경우: `tsc`

를 실행합니다.

이렇게 했을때 아무 오류가 없었다면, 축하드립니다! 성공적으로 모든 타입스크립트 파일을 자바스크립트로 변환하셨습니다!

## ts-node로 원하는 타입스크립트 파일 바로 실행하기

만약 전 단계에서 `ts-node`를 설치하셨다면, ts-node를 통해 바로 원하는 타입스크립트 파일을 실행 시킬 수 있습니다.

터미널에

- 그냥 설치 한 경우: 
	- npm: `npx ts-node <파일 이름>.ts`
	- pnpm: `pnpm ts-node <파일 이름>.ts`
	- yarn: `yarn ts-node <파일 이름>.ts`
- 글로벌로 설치한 경우: `ts-node <파일 이름>.ts`

를 실행하면 끝납니다!

(진짜 정말 죄송한데 다시 한번만 더 말씀드릴게요. `<파일 이름>`에는 반드시 타입스크립트 파일 이름을 적으셔야 합니다! 제발제발제발제발)

그러면 바로 콘솔에서 결과를 확인해 볼 수 있습니다.

## +) 감시 모드 설정해서 수정사항 바로바로 변환하기

이런 경우가 있을겁니다.

> "아... 파일 저장하고 컴파일 해야하는거 귀찮은데..."

정말 더럽게 귀찮죠? 하지만 그런 분들을 위해 아주 딱 좋은 기능이 하나 있습니다. 바로 '감시 모드' 죠!

터미널에

- 그냥 설치 한 경우: 
	- npm: `npx tsc -w` 혹은 `npx tsc --watch`
	- pnpm: `pnpm tsc -w` 혹은 `pnpm tsc --watch`
	- yarn: `yarn tsc -w` 혹은 `yarn tsc --watch`
- 글로벌로 설치한 경우: `tsc -w` 혹은 `tsc --watch`

를 실행하면 감시 모드를 켤 수 있습니다! 이렇게 관찰 모드를 실행시키면 파일에 변경사항이 있을때마다 바로바로 타입스크립트를 변환시킬 수 있습니다. 너무 편리하죠?
